# Projects, Labs, and Assignments

This workspace contains a collection of notebooks, documents, and files used for various coursework and personal experiments. Below is an overview of the main components organized by folder and file.

## Top-level files

- **filters.ipynb** - Jupyter notebook demonstrating image filtering techniques using `numpy` and `scipy`. Includes examples of convolution, edge detection, and custom kernels.
- **pil_numpy_matplotlib.ipynb** - Notebook exploring image processing with PIL, NumPy arrays, and visualization with Matplotlib. Contains tutorials on reading, transforming, and displaying images.
- **kitten.png**, **sakura.png** - Sample image files used within the notebooks for processing and visualization tasks.
- **ЛР3_Основы телевидения_Принципы формирования сигналов ЦТВ.docx** - Laboratory report on the principles of forming digital television signals.
- **ЛР5_Основы телевидения_Цветокоррекция.docx** - Laboratory report on color correction in television systems.

## `week_01_ml_intro` folder

This directory holds materials for the first week of a machine learning introductory course.

### Homework Subfolder

- **hw_1_game_of_thrones.ipynb** - Assignment notebook applying basic Python and data analysis skills to a dataset inspired by *Game of Thrones*. Includes exercises on data loading, cleaning, and visualization.


## Usage Notes

Open the notebooks with Jupyter (or VS Code's notebook interface) to run cells interactively. The image files are referenced within the notebooks and serve as examples for the various processing routines.
